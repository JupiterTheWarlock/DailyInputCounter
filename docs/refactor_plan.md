# 🔄 DailyInputCounter 重构计划 (MVP)

## 📝 重构背景

当前项目使用 Python + tkinter 实现，需要快速转向无依赖的独立应用。
核心需求：统计中英文输入比例，反映AI使用率（中文占比高表示AI使用率高）。

## 🎯 MVP目标（4小时完成）

1. **最小可用产品**:
   - 键盘监听 + 中英文统计
   - 简单的数据展示界面
   - 独立exe程序
   - 系统托盘显示

2. **核心指标**:
   - 中文字符数
   - 英文字符数
   - AI率 = 中文字符数 / (中文字符数 + 英文字符数) * 100%

## 🛠️ MVP技术选型

### 核心框架（追求简单快速）
- **Electron**: 最小配置
- **React**: 仅使用基础组件
- **electron-store**: 简单数据存储

### 必要依赖
- **iohook**: 键盘监听
- **electron-builder**: 打包工具

## 🚀 MVP功能模块

### 1. 键盘监听（1小时）
- 仅监听字符输入
- 中英文分类统计
- 实时计数

### 2. 数据存储（30分钟）
- 使用electron-store
- 仅存储日期和字符数
- 无需复杂查询

### 3. 界面（2小时）
- 系统托盘图标
- 简单统计面板
- AI率显示
  
### 4.输出报告
- 输出简单的每日中英文字符数报告 

### 4. 打包（30分钟）
- 单文件exe
- 基础安装程序
