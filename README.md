# 📊 DailyInputCounter - 程序员AI率统计器

一个用于统计程序员每日中英文键盘输入量的轻量级工具，通过计算每日输出字符数来反映工作产出的"AI率"。
中文越多，英文越少，就说明使用ai越多，手敲代码越少。

## 🎯 项目初衷

作为程序员，我们想知道自己每天到底输出了多少代码和文档。通过统计每日的中英文字符输入量，可以直观反映AI辅助程度：
- 英文输入多：可能在手写代码
- 中文输入多：可能在和AI对话

## ✨ 核心功能

- 🔍 **双语识别**: 自动区分中文汉字和英文字母
- 📈 **实时统计**: 后台静默统计，当日数据实时更新
- 💾 **轻量存储**: 每日仅存储两个数字：中文字符数+英文字符数
- 📊 **AI率计算**: 通过中英文字符比例计算AI使用率
- 🚀 **开机自启**: 自动启动，无需手动操作
- 🔐 **隐私保护**: 仅统计字符数量，不记录内容

## 💡 AI率分析

### 计算公式
```
AI率 = 中文字符数 / (中文字符数 + 英文字符数) * 100%
```

### AI率指标
- **高AI率**: > 70% (主要使用AI辅助)
- **中等AI率**: 30-70% (人机结合开发)
- **低AI率**: < 30% (主要手写代码)

### 字符识别规则
| 字符类型 | 识别范围 | 说明 |
|---------|---------|------|
| 中文字符 | \u4e00-\u9fff | 汉字、中文标点 |
| 英文字符 | a-zA-Z | 英文字母 |
| 其他 | 忽略 | 数字、符号不统计 |
